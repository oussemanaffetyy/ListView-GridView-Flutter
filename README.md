## Installation



### Dependency
Add the package as a dependency in your pubspec.yaml file.
```yaml
dependencies:
  flutter_launcher_icons: "^0.13.1"
```

### Import
Import the package in your code file.
```dart
import 'package:flutter_staggered_animations/flutter_staggered_animations.dart';
```
